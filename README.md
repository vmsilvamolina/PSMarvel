## Usage

### Installation

Install the module from the [PowerShell Gallery](https://www.powershellgallery.com/) with the command:

```powershell
Install-Module -Name PSMarvel
```

### Commands available

The following commands are available:

```powershell
Get-Command -module PSMarvel
```
